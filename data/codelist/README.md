# Code Lists for the Vehicle Dismantling Industry  
# Kodlistor för bildemonteringsbranschen

## English

### Background

Bildelsbasen Nordic AB has operated and maintained the vehicle dismantling industry’s shared code lists since their inception in the 1990s. These code lists form a common classification framework and include:

- **vehicle codes** (make, model, generation, model year)  
- **part codes** with generic, manufacturer-independent naming  
- **position codes**, defining where on the vehicle a part originates  
- **manufacturer codes**, listing part manufacturers to enable correct and consistent linkage  

The work has included:

- investigation and quality assurance of new codes  
- defining generic and function-based terminology that works independently of vehicle brand  
- maintaining structured groupings that support correct compatibility assessment  

The primary purpose of this structure is to enable accurate assessment of **which vehicle parts may be compatible**, to describe **where a part originates on the vehicle**, and to provide a common and stable foundation for information exchange across the industry.

---

### Use of the Code Lists

By using the code lists, the user undertakes to:

- keep the code lists up to date in accordance with updates provided by Bildelsbasen Nordic AB  
- use the code lists in their entirety and without modification  

It is **not permitted** to:

- modify the structure of the code lists  
- change code names or codes  
- add proprietary or custom codes  

This applies equally to vehicle codes, part codes, position codes, and manufacturer codes.

Any deviation from this may result in the code lists being fragmented into multiple versions. Over time, this can lead to serious data integrity issues, for example where parts are incorrectly classified, positioned, or linked to the wrong manufacturer in external systems or marketplaces.

The code lists are intended to function as a **shared reference**, not as a locally adaptable or proprietary rule set.

---

### Purpose

The overarching purpose of the code lists is to establish and maintain an **industry standard** that ensures all suppliers, systems, and marketplaces speak the same language and use the same structure.

This means that:

- data and code lists do not need to be converted between systems  
- no technical lock-in occurs in dismantling systems or marketplaces  
- customers can freely change system providers without extensive technical work  

The code lists are therefore intended to serve as an **open and neutral interface** that benefits the entire industry rather than individual actors.

---

### Responsibility

The user is responsible for:

- verifying that the code lists in use are current and correctly implemented  

If errors or inconsistencies are discovered, these must be reported to Bildelsbasen Nordic AB via email:  
**info@bildelsbasen.se**

All investigations, corrections, and any necessary adjustments are handled centrally to maintain long-term data quality and consistency.

---

### Cost

The code lists are managed, updated, and distributed by Bildelsbasen Nordic AB.

This work is carried out **free of charge**, with the purpose of:

- keeping the code lists open and accessible  
- enabling a supplier-independent interface  
- ensuring long-term stability and shared benefit for the entire vehicle dismantling industry  

The code lists are not intended to function as a competitive instrument, but as shared infrastructure.

## Svenska

### Bakgrund

Bildelsbasen Nordic AB har driftat bildemonteringsbranschens gemensamma kodlistor sedan starten på 1990-talet. Kodlistorna utgör ett gemensamt klassificeringsramverk och omfattar:

- **bilkoder** (fabrikat, modell, generation, årsmodell)  
- **delkoder** med generella och fabrikatsoberoende benämningar  
- **positionskoder**, som anger var på fordonet delen har sitt ursprung  
- **tillverkarkoder**, en lista över tillverkare av bildelar för korrekt och konsekvent koppling  

Arbetet har bland annat inneburit att:

- utreda och kvalitetssäkra nya koder  
- ta fram funktionella och generella termer som fungerar oberoende av fabrikat  
- förvalta strukturer som stödjer korrekt kompatibilitetsbedömning  

Syftet med denna struktur är att möjliggöra korrekt bedömning av **vilka bildelar som kan vara kompatibla**, att tydligt beskriva **delens position och ursprung**, samt att skapa en gemensam och stabil grund för informationsutbyte i branschen.

---

### Användning av kodlistorna

Genom att använda kodlistorna förbinder sig användaren att:

- hålla kodlistorna uppdaterade enligt de uppdateringar som tillhandahålls av Bildelsbasen Nordic AB  
- använda kodlistorna i sin helhet och utan förändringar  

Det är **inte tillåtet** att:

- förändra kodlistornas struktur  
- ändra benämningar eller koder  
- lägga till egna koder  

Detta gäller samtliga kodlistor, inklusive bilkoder, delkoder, positionskoder och tillverkarkoder.

Avvikelser från detta riskerar att leda till att kodlistorna splittras i flera versioner. I förlängningen kan detta orsaka allvarliga fel i dataflöden, exempelvis att delar klassificeras fel, får fel position eller kopplas till fel tillverkare i externa system eller marknadsplatser.

Kodlistorna är avsedda att fungera som en **gemensam referens**, inte som ett lokalt eller anpassningsbart regelverk.

---

### Syfte

Kodlistornas övergripande syfte är att etablera och upprätthålla en **branschstandard** där alla leverantörer, system och marknadsplatser använder samma språk och struktur.

Detta innebär att:

- data och kodlistor inte behöver konverteras mellan olika system  
- ingen teknisk inlåsning uppstår i vare sig demonteringssystem eller marknadsplatser  
- kunder och leverantörer fritt kan byta systemleverantör utan omfattande tekniskt arbete  

Kodlistorna ska därmed fungera som ett **öppet och neutralt gränssnitt** som gynnar hela branschen, inte enskilda aktörer.

---

### Ansvar

Användaren ansvarar själv för att:

- kontrollera att använda kodlistor är aktuella och korrekt implementerade  

Om fel eller oklarheter upptäcks ska detta rapporteras till Bildelsbasen Nordic AB via e-post:  
**info@bildelsbasen.se**

All hantering av korrigeringar, utredningar och eventuella justeringar sker centralt för att säkerställa fortsatt datakvalitet och konsekvens.

---

### Kostnad

Kodlistorna hanteras, uppdateras och distribueras av Bildelsbasen Nordic AB.

Detta arbete sker kostnadsfritt, i syfte att:

- hålla kodlistorna öppna och tillgängliga  
- möjliggöra ett fritt och leverantörsoberoende gränssnitt  
- säkerställa långsiktig stabilitet och gemensam nytta för hela bildemonteringsbranschen  

Kodlistorna är inte avsedda att fungera som ett konkurrensmedel, utan som en gemensam infrastruktur.
